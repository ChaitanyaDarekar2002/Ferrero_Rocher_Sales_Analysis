# 🍫 Ferrero Rocher Sales Data Analysis

## 📌 Project Overview

This project analyzes Ferrero Rocher sales data using Python, SQL Server, and Power BI. The objective is to clean, analyze, visualize, and transform raw sales data into meaningful business insights through an end-to-end data analytics workflow.

---

## 🛠️ Tools & Technologies

- Python
- SQL Server
- Power BI
- Pandas
- NumPy
- Matplotlib
- Microsoft Excel
